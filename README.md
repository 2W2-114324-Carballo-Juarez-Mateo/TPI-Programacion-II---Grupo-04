# 🚗 Sistema de Gestión de Alquiler de Vehículos

Proyecto final desarrollado en el marco de la Tecnicatura Universitaria en Programacion, orientado a la **gestión integral de vehículos de alquiler temporario**.
El sistema permite administrar vehículos, reservas, alquileres, pagos y disponibilidad, centralizando la información y facilitando la operación del negocio.

## 📌 Descripción general

El objetivo del proyecto es brindar una solución informática que permita:

- Registrar y administrar **vehículos disponibles para alquiler**

- Gestionar **reservas y alquileres activos**

- Registrar **pagos asociados a cada alquiler**

- Consultar **disponibilidad de vehículos** según fechas

- Centralizar la información de forma segura y estructurada

El sistema fue desarrollado bajo una **arquitectura cliente-servidor**, separando responsabilidades entre base de datos, backend y frontend.

## 🧱 Arquitectura del sistema

El proyecto está compuesto por tres capas principales:

### 🗄️ Base de Datos

- **Motor:** SQL Server

- **Diseño:** Modelo relacional normalizado

- **Responsabilidades:**

  - Persistencia de datos

  - Integridad referencial

  - Gestión de relaciones entre entidades (vehículos, alquileres, pagos, reservas, etc.)

### ⚙️ Backend

- **Lenguaje:** C#

- **Entorno:** Visual Studio

- **Tipo:** API REST

- **Responsabilidades:**

  - Lógica de negocio

  - Exposición de endpoints

  - Validaciones

  - Comunicación con la base de datos

### 🌐 Frontend

- **Editor:** Visual Studio Code

- **Tecnologías:**

  - HTML

  - CSS

  - JavaScript

  - Bootstrap (estilos y componentes)

- **Responsabilidades:**

  - Interfaz de usuario

  - Consumo de la API REST

  - Visualización y carga de datos

## 🧩 Funcionalidades principales
